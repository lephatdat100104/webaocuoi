# 🎓 Web Áo Cưới

## 📌 Giới thiệu
Đây là project website áo cưới giúp người dùng:
- Xem sản phẩm áo cưới
- Đặt dịch vụ
- Thanh toán online (VNPAY)

---

## 🛠 Công nghệ sử dụng
- Backend: ASP.NET Core
- Database: SQL Server
- Thanh toán: VNPAY
- Frontend: HTML, CSS, JavaScript

---

## 🚀 Chức năng chính

### 🔐 Authentication
- Đăng ký
- Đăng nhập

### 🛍 Sản phẩm
- Xem danh sách áo cưới
- Xem chi tiết sản phẩm

### 💳 Thanh toán
- Tích hợp VNPAY
- Xử lý thanh toán online
